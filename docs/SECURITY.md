# Security Policy

## Privacy Concerns

Please forward all privacy concerns to [gregory@earth-app.com](mailto:gregory@earth-app.com). You can request a deletion of your data anytime on our application's interface.

## Reporting a Vulnerability

You can report minor vulnerabilities by creating an issue, or if you can remedy the problem, create a pull request. For major vulnerabilities please contact [gregory@earth-app.com](mailto:gregory@earth-app.com).
